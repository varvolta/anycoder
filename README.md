# **Convert any js code to string and vice versa.**

Classes, Functions, Objects, Arrays and everything else.

<br />

# Installation
```
npm i anycoder
```

<br />

# Usage
```js
import AnyCoder from 'anycoder'

// encode with
const string = Anycoder.encode(Class)

// decode with
const Class = Anycoder.decode(string)

```
